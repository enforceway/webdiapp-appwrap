# webdiapp-appwrap客户端 README
  webdiapp-appwrap 是手机客户端程序，目前氛围安卓端手机程序以及 IOS 端手机程序。
  主要作用是客户端获取服务端数据并以列表形式在手机端，以分析域名对数据获取的影响的影响。
